# [stone.cn.mt](https://stone.cn.mt)

This is Stone's Site written in Next.js + TypeScript.

## Deploy

```bash
git clone https://github.com/stone2China/stone’web.git
cd nocp.space
npm i
npm run build
npm run start # http://localhost:3000
```

## Development

```bash
npm run dev # http://localhost:3000
```
